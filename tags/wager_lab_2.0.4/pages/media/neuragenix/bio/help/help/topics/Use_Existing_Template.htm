Use an Existing Template

 

  1. Click on the Integration tab.

  2. Click on the Use Existing Template option listed on the menu bar.

  3. Select your Template Name from the drop down menu.

  4. Click on the Next button at the bottom of the screen.

  5. You will be taken to the Upload File screen.

  6. Enter the File Name, or, click on the Browse button to find the file.

  7. Click on the Submit button at the bottom of the screen.
