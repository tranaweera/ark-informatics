Patient Management

The patient module captures and manages the patient record, consent details and appointments.

The patient record is required before capturing biospecimen and study details. The following functionality is available:

  • Create New Patient

  • Search Existing Patients

  • View Patient Details

  • Edit Patient Details

  • Delete Patient Details

  • Create Patient Consent

  • View Patient Consent

  • Edit Consent Details

  • Withdraw Consent

  • View Consent Forms

  • View Appointments

  • Create New Appointment

  • Edit Appointment

  • Cancel Appointment
