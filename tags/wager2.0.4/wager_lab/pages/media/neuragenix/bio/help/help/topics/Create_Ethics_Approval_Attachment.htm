Create Ethics Approval Attachment

 

  1. View the study record (See Search Study).

  2. Click on the Ethics Approval link. You will be taken to the Ethics Approval for Study screen.

  3. Click on the Add File option next to the appropriate Approval listed. You will be taken to the Add Ethics File screen.

  4. Enter the File Name in the appropriate field, if these details are known. Alternatively, click on the Browse button. You will be taken to File Select Dialogue Box.

  5. Double Click on the selected file. The file name will appear in the field next to the browse button.

  6. Click on the Save File button. You will be taken to the Ethics Approval for Study screen.
