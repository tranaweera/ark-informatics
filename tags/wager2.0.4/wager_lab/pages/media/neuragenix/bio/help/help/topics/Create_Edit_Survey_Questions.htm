Create/Edit Survey Questions

 

  1. View the Survey record. (See Search Survey).

  2. Click on the Add/Edit Questions button located below the Survey Description field. You will be taken to the Add Question to “Survey Title” screen.

  3. Click on the Add Question button.

  4. Enter the available details into the appropriate fields. The details entered will be displayed on the screen in those fields.

  5. Click on the Save button below Add/Edit Questions field. The edited information will be saved and displayed in the Question Order field.

To add a page break, click on Add Page Break button located below Question Order field.

To add a title or comment, click on the Add Title/Comment button located below the Question Order field.
