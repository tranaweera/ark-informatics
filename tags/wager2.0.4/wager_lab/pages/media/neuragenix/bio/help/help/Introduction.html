User's Guide Introduction

This user's guide contains the following parts:
About The Software A description of the software.
About The Installation A general description of the installation.
System Overview A general overview of the help system. Describes a range of functionalities related to the application.
System Requirements A brief description of the system requirements.
Login Information A guide for the users' security. Describes how to login and logout from the system.
Account Information A guide for the users' accounts. Describes how to create new account, to delete and to update account.
Record Animal Information A guide for the users keeping animal's records. Describes how to add new animal record, and to update record.
Treament History Information A guide for the users keeping treatment history's records of animals and chemical(s) used for particular treatment. Describes how to record treatment history and chemical used for the specific treatment such as adding, deleting, and updating record.
Event History Information A guide for the users keeping event history's records including paddock history and fate history of animals. Describes how to record paddock and fate history such as adding, deleting, and updating record.
Record Mating A guide for the users keeping mating history's records of animals. Describes how to add new mating record and update record.
Record Previous Owner A guide for the users keeping previous owner's information. Describes how to add new previous owner record and update record.
Record Veterinarian A guide for the users keeping veterinarian's information. Describes how to add new veterinarian record and update record.
Record Paddock A guide for the users keeping paddock's information. Describes how to add new paddock record and update record.
Record Stud A guide for the users keeping stud's information. Describes how to add new stud record and update record.
Record Chemical A guide for the users keeping chemical's record of treatment which an animal is given. Describes how to add new chemical record and update record.
Seach Facilities A guide for the users searching animal's information in the various ways. Describes how to make query about animal's information by using single search or combination search.
Wizard Information A guide for the users carrying out various wizards. Describes how to perform a task step by step such as, adding new treatment, new paddock for a group of animals, and either adding new mating for a group of dams, and weighing for a group of animals.
Print Report Facilities A guide for the users printing various reports on animal data. Describe how to print reports, such as pedigree report, herd list and notification form report.
Database Facilities A guide for the users dealing with database facilities. Describe how to back up and restore database from internal and external sources.
